PiTrailCam
Raspberry Pi based Trail Camera with time-lapse, video, and upload functionality (dropbox).

Hardware:
-Raspberry Pi Model A, A+, B, B+, Pi 2, or Pi3 (pi zero will not work as it does not have the camera option)
-SD Card
-Raspberry Pi Camera
-PIR Sensor
-Enclosure (optional)
-Wifi Module
-Battery Backup

Software:
-Raspbian (latest)
