PiTrailCam
==========
Raspberry Pi based Trail Camera with time-lapse, video, and upload functionality (dropbox).

###Hardware:
- Raspberry Pi Model A, A+, B, B+, Pi 2, or Pi3 ,Pi Zero v1.3
- SD Card
- Raspberry Pi Camera
- PIR Sensor
- Enclosure (optional)
- Wifi Module
- Battery Backup
- Keyboard (initial setup)
- Mouse (initial setup)

###Software:
- Raspbian (latest)
